GET

Get the IP-address for example.com
https://www.r53.be/get/example.com

You may specify IP version with the v query
https://www.r53.be/get/example.com?v=ipv4
https://www.r53.be/get/example.com?v=ipv6
If v is omitted or unknown then IPv6 is returned if it exists, else if IPv4 is returned if it exists.

If you query for something that does not exists in the database, then an empty string is returned.
